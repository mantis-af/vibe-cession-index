# Vibe-cession Index — Design Spec

## Overview

A weekly metro-level economic sentiment index that composites ~8 behavioral signals into a single 0-100 score per metro. The core product is a professional dashboard showing the index, its components, and the "Vibes Gap" (behavioral index vs official statistics).

## Architecture

### Frontend (Next.js 15 + TypeScript)
- **Framework**: Next.js 15 App Router, deployed to Vercel
- **Styling**: Tailwind CSS + shadcn/ui components
- **Charts**: Recharts (composable, React-native charting)
- **Data**: Static JSON files generated by Python pipelines, imported at build time via `src/data/`

### Data Pipelines (Python)
- Standalone Python scripts in `pipelines/`
- Each source has its own fetcher module
- `compute_index.py` normalizes signals to Z-scores, computes composite, outputs JSON
- Designed to run on a cron (GitHub Actions or local)

### Data Flow
```
Python pipelines → JSON files → Next.js static import → Dashboard
```

## MVP Scope

### 20 Target Metros
NYC, LA, Chicago, Houston, Phoenix, Philadelphia, San Antonio, San Diego, Dallas, San Jose,
Austin, Jacksonville, San Francisco, Columbus, Charlotte, Indianapolis, Seattle, Denver, DC, Nashville

### MVP Signals (4 of 8, all free)
1. **Google Trends Anxiety Index** — distress vs aspiration search ratio (pytrends)
2. **Initial Unemployment Claims** — DOL weekly state-level data
3. **Housing Inventory + Days on Market** — Redfin Data Center CSV
4. **BLS Local Unemployment (LAUS)** — monthly metro rates via BLS API

### Deferred Signals (add in iterations)
- Job postings velocity (Indeed Hiring Lab)
- Small business openings/closings (Yelp)
- Wage-to-rent ratio
- Restaurant reservation volume

## Index Computation
- Each signal → Z-score relative to metro's own 2-year history
- Weighted composite (equal weights for MVP, refine with backtesting)
- Rescaled to 0-100 for readability
- "Vibes Gap" = Behavioral Index - Official Index (BLS/CPI benchmark)

## Dashboard Pages
1. **Overview** — US map or grid of all 20 metros with index scores, color-coded
2. **Metro Detail** — drill into a single metro: index trend, component breakdown, vibes gap chart
3. **Compare** — side-by-side metro comparison

## UI Design Decisions
- Dark theme (professional, data-dense feel)
- Monospace accents for numbers
- Color scale: deep red (0, severe distress) → amber (50, neutral) → green (100, thriving)
- Minimal chrome, maximum data density
- Responsive but desktop-first (this is an analyst tool)
